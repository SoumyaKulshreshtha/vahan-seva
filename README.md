# Vahan Seva (Rural Bike Service)

A mobile-first platform connecting rural bike owners with verified local mechanics.

## 🚀 How to Run

### Option 1: Client Only (Quickest)
1.  Open `index.html` in your browser.
2.  Allow Location access to see nearby mechanics.
3.  *Note: This uses mock data since the backend isn't running.*

### Option 2: Full Stack (With Backend)
Since you have Node.js installed:

1.  Open your **Command Prompt** or **Terminal**.
2.  Navigate to this folder:
    ```bash
    cd "c:\Users\aksha\OneDrive\Desktop\managment project\rural-bike-service"
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```
4.  Start the server:
    ```bash
    npm start
    ```
    *(You should see "✅ Backend running on http://localhost:3000")*

5.  Now open `index.html` in your browser. The app will fetch real data from the database!

## Features
-   **Vahan Seva UI**: Premium Blue/Green theme.
-   **Real-Time Geolocation**: Sorts mechanics by proximity to you.
-   **Backend Integration**: Supports SQLite database for persistence.
